Promjene stringova

  • razdvajanje (split())
  • zamjena (sub,subn())

  • grupe dohvaanja
  • povratnost
  • imenovanje grupa

Python | Regex | split

re.split(pattern,string,maxsplit=0,flags=0)
  • funkcija koja razdvaja string prema pojavi podudaranja s pattern
  • opcija maxsplit kae koliko najvie elimo podjela
  • ukoliko koristimo regex grupe dohvaanja (), tekst te grupe se isto pojavljuje u podjeli.

Python | Regex | sub, subn

re.sub(pattern,repl,string,count=0,flags=0)
  • vrati string vraen zamjenom prve (s lijeva na desno) nepreklapajue podudarajue pojave uzorka pattern u string s parametrom repl
  • ukoliko uzorak nije naen vrati nepromijenjen string
  • parametar repl moe biti string ili funkcija
  • opcionalni parametar count predstavlja max. broj zamjena
re.subn(pattern,repl,string,count=0,flags=0)
  • ista funkcionalnost kao sub() osim to vraa dvojku izmjenjenog stringa i broj uinjenih zamjena

Python | regex | grupe dohvaanja

Grupe dohvaanja unutar regex uzorka, u oznaci (), slue kasnijem dohvaanju preko:

  • group(num=x) metode iz objekta sparivanja matchObj gdje je x redni broj grupe
  • povratno referenciranje ili iskljuivanje iz dohvaanja
  • mogu biti imenovane
In[4]:
# regex | grupe dohvaanja
import re
# Razlika uzoraka do na zagrade?
# 1. standardno podudaranje uzoraka bez grupa
tekst = "Srpanj 24, Kolovoz 9, Prosinac 12, Travanj D"
regex = r"[a-zA-Z]+ \d+"
matches = re.findall(regex, tekst)
print(matches)
# 2. dohvaanje posebnog dijela podudarenog uzorka
regex = r"([a-zA-Z]+) \d+"
matches = re.findall(regex, tekst)
print(matches)

# search
matchObj = re.search(r'([a-zA-Z]+) (\d+)',tekst)
matchObj.group()   # dohvaanje bez grupe
matchObj.group(1) # dohvacanje 1. grupe 
matchObj.group(2) # dohvacanje 2. grupe

['Srpanj 24', 'Kolovoz 9', 'Prosinac 12']
['Srpanj', 'Kolovoz', 'Prosinac']

Python | regex | povratnost (engl. backreference)

U funkciji zamjene sub od koristi je ponekad koristiti povratnost:

  • koritenje reference na prijanju grupu iz regex uzorka
  • oznaka: \i gdje i redni broj grupe u pattern ukoliko se dohvaa unutar uzorka, inae \g<i>
In[5]:
# primjer 1: povratnost
test = 'Python&Pails'
pattern = r'([Pp])ython&\1ails'
re.match(pattern,test).group()

# primjer 2: MarkDown 2 HTML konverzija
tekst = 'zamislite bolju *Hrvatsku*, *arobnu* Hrvatsku'
pattern = r'\*(.*?)\*' # nepohlepni pristup pronalazenja *izraz*
# \g<1> predstavlja referencu na 1. grupu: (.*?)
re.sub(pattern,r'<b>\g<1></b>',tekst,count=1) # count predstavlja broj ostvarenih zamjena
Out[5]:
'Python&Pails'
Out[5]:
'zamislite bolju <b>Hrvatsku</b>, *arobnu* Hrvatsku'

izuzimajue grupe dohvaanja

Grupe dohvaanja moemo eksplicitno izuzeti u oznaci

(?:pattern)

to znai:

  • potraga pronalazi pattern unutar stringa, ali ne vraa podudarajui podstring za tu grupu
  • grupa se ne moe referencirati
In[113]:
# dohvacanje preko findall 
tekst = 'abbaabbba'
re.findall(r'(a((a+)|(b+)))',tekst) # ukupno 3 grupe dohvaanja

# iskljuivanje grupe:
# primjer 1
re.findall(r'(a((?:a+)|(?:b+)))',tekst) # ukupno jedna grupa dohvaanja
# primjer 2
re.findall(r'(a(a+))',tekst) 
re.findall(r'a(?:a+)',tekst)
Out[113]:
[('abb', 'bb', '', 'bb'), ('aa', 'a', 'a', '')]
Out[113]:
[('abb', 'bb'), ('aa', 'a')]
Out[113]:
[('aa', 'a')]
Out[113]:
['aa']

imenovanje grupa

  • grupe dohvaanja mogu se imenovati s oznakom (?P<name>) gdje name predstavlja ime te grupe
  • grupa se dohvaa s (?P=name) i predstavlja novu grupu s uzorkom grupe name
In[6]:
#imenovane grupe
import re

tekst = 'Ovo je neki tekst !?. sa znakovima interpunkcije.'

for pattern in [ r'^(?P<prva_rijec>\w+)',
                   r'(?P<zadnja_rijec>\w+)\S*$',
                   r'(?P<t_rijec>\bt\w+)\W+(?P<neka_rijec>\w+)',
                  r'(?P<zavrsava_t>\w+t)\b'
                ]:
    
    match = re.search(pattern,tekst)
    print('\nregex: "%s": '.rjust(10) % pattern)
    print('\tgrupe:\t\t '.rjust(10),  match.groups())
    print('\trjecnici imena:\t '.rjust(8),  match.groupdict())

regex: "^(?P<prva_rijec>\w+)": 
	grupe:		  ('Ovo',)
	rjecnici imena:	  {'prva_rijec': 'Ovo'}

regex: "(?P<zadnja_rijec>\w+)\S*$": 
	grupe:		  ('interpunkcije',)
	rjecnici imena:	  {'zadnja_rijec': 'interpunkcije'}

regex: "(?P<t_rijec>\bt\w+)\W+(?P<neka_rijec>\w+)": 
	grupe:		  ('tekst', 'sa')
	rjecnici imena:	  {'t_rijec': 'tekst', 'neka_rijec': 'sa'}

regex: "(?P<zavrsava_t>\w+t)\b": 
	grupe:		  ('tekst',)
	rjecnici imena:	  {'zavrsava_t': 'tekst'}

Primjer 3*: Harmonija finskih samoglasnika

Sufiksi u imenicama finskog jezika imaju najee 2 morfema koje ovise o samoglasniku koji je vezan na korijen rijei:

  • sufiks -lla dolazi u svim sluajevima kada korijen rijei sadri u,o i a i nema infiksa y, , or
  • sufiks -ll inae
In[103]:
back_vowel = pynini.union("u", "o", "a")
neutral_vowel = pynini.union("i", "e")
front_vowel = pynini.union("y", "", "")
vowel = pynini.union(back_vowel, neutral_vowel, front_vowel)

# naglasci
archiphoneme = pynini.union("A", "I", "E", "O", "U")
# suglasnici
consonant = pynini.union("b", "c", "d", "f", "g", "h", "j", "k", "l", "m", "n",
                         "p", "q", "r", "s", "t", "v", "w", "x", "z")
# alfabet
sigma_star = pynini.union(vowel, consonant, archiphoneme).closure()
# sufiks na kraju imenice
adessive = "llA"
# infiksi
intervener = pynini.union(consonant, neutral_vowel).closure()
# FST
adessive_harmony = (pynini.cdrewrite(pynini.cross("A", "a"),
                                     back_vowel + intervener, "", sigma_star) @
                    pynini.cdrewrite(pynini.cross("A", ""), "", "", sigma_star)
                   ).optimize()


def make_adessive(stem):
    ur = stem + adessive
    sr = pynini.compose(ur,adessive_harmony)
    return sr.string()
